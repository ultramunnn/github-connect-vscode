Hallo ini adalah cara connect github ke vscode

Berikut caranya connect ke github di vscode:

  1. Buat akun di github
  2. Buat repository di github
  3. Install extension Github Pull Requests
  4. Install extension Github repository
  5. Install git-bash
  6. Buka terminal git-bash
  7. Ketik git config --global user.email akunGitHub@gmail.com
  8. Ketik git config --global user.name usernamemu di github
